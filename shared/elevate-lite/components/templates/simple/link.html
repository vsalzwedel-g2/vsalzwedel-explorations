Usage Guidelines

  • Standard vs Subtle: Use standard (blue) for primary navigation, subtle (gray) for footer or secondary links
  • Underlines: Add underlines in dense text to improve scannability and accessibility
  • Size Selection: Match link size to surrounding text (md = body, sm = small text, xs = captions/footers)
  • Semantics: Use <a> for navigation, <button class="elv-link-button"> for actions
  • External Links: Add rel="noopener noreferrer" for security when using target="_blank"

Accessibility Notes

  • Focus Indicators: 2px purple outline appears on keyboard focus (never remove it!)
  • Color Independence: Consider adding underlines when color alone conveys link status
  • Descriptive Text: Link text should make sense out of context (avoid "click here" or "learn more")
  • External Links: Consider indicating external links with an icon or text "(opens in new tab)"
  • Skip Links: For accessibility, add skip-to-content links at the top of pages

Design System Specs

Standard Color #0073f5 (blue-100 / text-link)
Hover Color #005bc2 (blue-120 / text-link-hover)
Visited Color #2e90ff (blue-80 / text-link-visited)
Subtle Color #6f6d78 (neutral-70 / N70)
Focus Outline 2px solid #5746b2 (P100) with 2px offset, border-radius 8px (sm/md) or 12px (xs)
Sizes MD (16px), SM (14px), XS (12px)
Variants Standard (blue), Subtle (gray), with/without underline
