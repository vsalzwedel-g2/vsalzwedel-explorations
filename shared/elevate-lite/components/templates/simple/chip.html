Chip

Interactive filter and selection component · See Elevate Lookbook for full documentation

Component Type: Interactive selection/filter (not semantic badge). Supports single-select, multiselect, and dismissible variants with hover and selected states.

Basic (Single Select)

Click to toggle selection state. Base is neutral, hover shows purple preview, selected persists purple.

<button class="elv-chip elv-chip--md" data-kind="single_select" data-selected="false">
  <span class="elv-chip__label">Filter Name</span>
</button>

Shows close icon for removal. Click anywhere to select, click close icon to dismiss. Fades out before removal.

Usage Guidelines

  • Single Select: Use for exclusive filter options where only one can be active (e.g., view mode toggles)
  • Multiselect: Use for cumulative filters where multiple can be active (e.g., category tags, feature filters)
  • Dismissible: Use for applied filters or tags that can be removed (e.g., search refinements, active selections)
  • Icon sizes: xs/sm chips use 12px icons, md chips use 16px icons
  • Animation: Icon width transitions smoothly, dismissible chips fade out before removal
