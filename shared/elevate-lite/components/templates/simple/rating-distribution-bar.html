Usage Guidelines

  • Purpose: Display rating distribution across 5 star levels in review summaries
  • Prefix: Use star icons (1-5 stars) to show rating level. Can also use text labels.
  • Progress Bar: Width represents proportion of total reviews at that rating level
  • Suffix Count: Show actual number of reviews (not percentage) for transparency
  • Ordering: Always display 5 stars to 1 star (descending order)
  • Interactive Use: Can wrap in label with radio input to filter reviews by rating
  • Size Selection: Use sm for compact contexts, lg for prominent displays

Accessibility Notes

  • Static Display: Add role="group" and aria-label="Rating distribution" to container
  • Interactive Filter: Use semantic <label> wrapping <input type="radio">
  • Focus Visible: Component shows visible focus ring via :focus-visible on radio input
  • Screen Reader: Each bar should describe "X star rating: Y reviews"
  • Keyboard Navigation: Interactive version supports arrow keys for radio group navigation
<!-- Static (non-interactive) -->
<div role="group" aria-label="Rating distribution">
  <div class="elv-rating-distribution-bar elv-rating-distribution-bar--sm"
       role="img" aria-label="5 stars: 2,145 reviews">
    <!-- content -->
  </div>
</div>

<!-- Interactive filter -->
<fieldset>
  <legend class="elv-sr-only">Filter reviews by rating</legend>
  <label class="elv-rating-distribution-bar elv-rating-distribution-bar--sm">
    <input type="radio" name="rating" value="5"
           aria-label="Filter by 5 star reviews (2,145 reviews)">
    <!-- content -->
  </label>
</fieldset>

Design System Specs

Sizes SM (4px/8px padding, 12px gap), LG (8px padding, 12px gap)
Progress Bar Height 8px (sm size from progress_bar component)
Progress Fill Color #ff492c (Rorange / brand color)
Progress Track Color #f2f2f3 (N10)
Star Size (prefix) 20px (md size from star_rating component)
Count Font (suffix) SM: 14px/400, LG: 14px/600
Hover Background #f5f3fd (primary-5) when interactive
Focus Outline 2px solid #c3bde5 (purple-40)
Border Radius 4px (sm from radius scale)
