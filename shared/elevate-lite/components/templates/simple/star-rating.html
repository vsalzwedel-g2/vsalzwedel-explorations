Usage Guidelines

  • Size Selection: Use sm for compact cards, md for standard displays, lg for prominent ratings
  • Review Count: Always include review count to build trust and provide context
  • Rounding: Round to nearest 0.5 (half-star increment)
  • Format: Display as "X out of 5" for clarity
  • Readonly: This component is display-only (not interactive rating input)

Accessibility Notes

  • ARIA: Add role="img" and aria-label to star container
  • Text Alternative: The "X out of 5" text provides screen reader context
  • Decorative: Stars are decorative when text value is present
  • Color Independence: Rating value doesn't rely solely on color (text included)

Design System Specs

Star Sizes SM (16px), MD (20px), LG (24px)
Star Gap 0 (stars adjacent, no gap)
Full Star Color #FF492C (Rorange)
Empty Star Color #DFDFE2 (N20)
Text Color #4c4b53 (text-subtle)
Text Sizes SM: 12px, MD: 14px, LG: 16px
Font Weight 300 (light) for rating text
