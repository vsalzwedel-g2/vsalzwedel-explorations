Usage Guidelines

  • Size Selection: Use xs/sm for inline loading, md for cards, lg for full-page loading
  • Color Choice: Primary for branded contexts, neutral for subtle loading, inverted for dark backgrounds
  • Placement: Center loaders in their container for best visual balance
  • Timing: Show loader after 300-500ms delay to avoid flashing on fast loads
  • Context: Always pair with descriptive text when possible ("Loading...", "Saving changes...")

Accessibility Notes

  • ARIA: Add role="status" and aria-live="polite" to loading containers
  • Screen Readers: Include visually-hidden text like "Loading" for screen reader users
  • Focus Management: Prevent interaction with loading content by disabling or removing from tab order
  • Animation: Respect prefers-reduced-motion by pausing animation for users who need it

Reduced Motion Support

For users who prefer reduced motion, pause the animation 

@media (prefers-reduced-motion: reduce) {
  .elv-spin-loader {
    animation: none;
    /* Show static version or pulse instead */
  }
}
