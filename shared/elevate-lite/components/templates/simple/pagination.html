Pagination

Navigate through multi-page datasets and lists. See Elevate Lookbook for full documentation.

Simple pagination with all pages visible. Current page is highlighted in purple.

Usage Notes

  • Keyboard Navigation: Arrow keys navigate between pages when pagination is focused, Enter/Space activates
  • ARIA Labels: Always include descriptive aria-label attributes for screen readers
  • Current Page: Use aria-current="page" and elv-pagination__btn--current class
  • Disabled State: Use disabled attribute for Previous/Next when at first/last page
  • Truncation: Show first, last, current, and 1-2 adjacent pages. Use ellipsis (…) for gaps
  • Ellipsis: Use aria-hidden="true" on ellipsis spans (not interactive)
  • Mobile: On mobile (<640px), only show nav buttons, current, first, and last pages
  • Links vs Buttons: Use <button> for JavaScript navigation, <a> for server-side pagination with href
  • URL Updates: For SPA/client-side routing, update URL with pushState or hash on page change
