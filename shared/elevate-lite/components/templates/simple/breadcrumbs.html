Usage Guidelines

  • Current Page: Always show as non-clickable span with aria-current="page"
  • Parent Links: All ancestors should be clickable links
  • Separators: Use chevron-right icon between items (no separator after current page)
  • SEO: Include schema.org markup for search engine indexing
  • Placement: Typically placed at top of page, above main heading
  • Length: Keep to 2-5 levels deep; consider truncation for very long paths

Accessibility Notes

  • Semantic HTML: Use <nav> with aria-label="Breadcrumb"
  • List Structure: Use <ol> with role="list" to preserve semantics
  • Current Page: Mark with aria-current="page" and disable pointer events
  • Focus States: All links have visible focus indicators
  • Link Text: Use meaningful labels (not "Back" or "Previous")
  • Landmarks: Breadcrumbs are a navigation landmark for screen readers
