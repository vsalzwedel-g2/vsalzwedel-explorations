Accessibility Notes

  • Alt Text: Always provide product name as alt text (e.g., "Slack", "Salesforce")
  • Semantic HTML: Use <img> tag for product logos (not decorative background images)
  • Placeholder Icon: Should have descriptive meaning (e.g., generic product/software icon)
  • Focus States: If product avatar is clickable/interactive, add visible focus ring

Design System Specs

Sizes XXL (168px), XL (144px), LG (104px), MD (80px), SM (64px), XS (48px), XXS (24px)
Shape Square with rounded corners (not circular)
Border 1px solid var(--border-light)
Border Radius XXL/XL: 8px, LG/MD/SM/XS: 4px, XXS: 2px
Padding XXL/XL: 8px, LG: 4px, MD and smaller: 0px
Shadow Only XXL has shadow-1, others have no shadow
Background var(--bg-neutral-0) white
Image Fit object-fit: contain (not cover - preserves logo aspect ratio)
Display Types Product logo (image) or placeholder icon

Usage Guidelines

When to Use Product Avatar

  • Product listings, comparison tables, recommendation grids
  • Product detail page headers
  • Search results showing software products
  • Alternative product suggestions
  • Vendor/product identification in reviews or comparisons

Key Differences from User Avatar

  • Shape: Square/rounded-square (product) vs. circular (user)
  • Fit: contain (product logos preserve aspect ratio) vs. cover (user photos fill circle)
  • Purpose: Brand/product identity vs. human identity
  • Placeholder: Generic product icon vs. user icon or initials

Size Selection

  • XXL/XL: Product detail page headers, hero sections
  • LG: Product cards, featured listings, grid views
  • MD: List items, search results (most common)
  • SM/XS: Compact lists, tags, inline mentions
  • XXS: Dense tables, minimal UI contexts
