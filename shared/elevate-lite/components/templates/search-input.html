Specifications

  • Input type: type="search" for semantic HTML
  • Base class: elv-input (provides border, background, transitions from Elevate)
  • Height: elv-h-9 (36px / 2.25rem) to match chips and dropdown
  • Border radius: elv-rounded-full (fully rounded ends, matches Elevate md search field)
  • Icon size: elv-icon-sm (20×20px)
  • Icon position: elv-left-3 (12px from left), vertically centered
  • Input padding: elv-pl-10 (40px) to clear icon
  • Text size: elv-text-sm (14px)
  • Icon color: elv-fill-neutral-80 (#78716C)
  • Accessibility: Always include aria-label attribute
  • Focus state: Light lavender outline (3px solid #c3bde5) with 1px offset - automatic from elv-input
  • Note: Elevate's standard search input uses elv-input--md (40px), but we override with elv-h-9 (36px) for visual alignment with other controls
