Accessibility Notes

  • Labels: Always wrap input and label in a <label> element for clickability
  • Radio Groups: Use <fieldset> and <legend> for radio button groups
  • Checkbox Groups: Use role="group" and aria-labelledby for checkbox groups
  • Focus State: 3px solid #c3bde5 (light purple P40) outline offset by 1px — CRITICAL for keyboard navigation
  • Keyboard Navigation: Space to toggle checkbox/radio, Tab to move between inputs
  • Error States: Use aria-invalid="true" and aria-describedby for error messages
  • Helper Text: Use aria-describedby to link helper text to input
  • Required Fields: Use required attribute for form validation
  • Touch Targets: Minimum 20px × 20px, use 24px for better mobile accessibility (--lg variant)
  • Indeterminate: Set via JavaScript: checkbox.indeterminate = true
