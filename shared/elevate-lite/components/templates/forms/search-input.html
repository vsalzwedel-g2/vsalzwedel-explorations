Accessibility Notes

  • ARIA Labels: Always include aria-label="Search" on the input
  • Clear Button: Include aria-label="Clear search"
  • Focus State: 3px light purple outline (#c3bde5) with 1px offset — CRITICAL for accessibility
  • Keyboard: Enter to submit, Esc to clear
  • Type: Use type="search" for semantic HTML
