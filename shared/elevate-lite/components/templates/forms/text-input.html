Accessibility Notes

  • Labels: Always pair inputs with labels using for and id attributes
  • ARIA Labels: Include aria-label when label is not visible
  • Error States: Use aria-invalid="true" and aria-describedby for error messages
  • Required Fields: Use required attribute and aria-required="true"
  • Focus State: 3px light purple outline (#c3bde5) with 1px offset — CRITICAL for accessibility
  • Keyboard Navigation: Tab to move between inputs, Shift+Tab to move backward
  • Password Toggle: Include aria-label="Toggle password visibility" on toggle button
  • Helper Text: Use aria-describedby to link helper text to input
  • Touch Targets: Minimum 40px height, use 48px for better mobile accessibility
