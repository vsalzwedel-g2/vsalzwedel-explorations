Accessibility Notes

  • Labels: Always pair upload zone with a visible label using for and id
  • ARIA Labels: Include aria-label on hidden file input and remove buttons
  • Keyboard Access: Click zone or press Enter/Space to trigger file dialog
  • Focus Indicator: Visible outline on keyboard focus
  • Error States: Use aria-invalid="true" and aria-describedby for error messages
  • Screen Readers: File list updates are announced when files are added/removed
  • Required Fields: Use required and aria-required="true"
  • Helper Text: Clearly indicate accepted file types and size limits
  • Remove Buttons: Include descriptive aria-label="Remove [filename]"
