Usage Guidelines

  • Purpose: Use for table of contents or anchor links to page sections
  • vs Tabs: Tabs switch content views; index nav scrolls to content on same page
  • Active State: One item should be active at a time, tracking current section
  • Anchors: Always use anchor links (href="#section-id") for proper navigation
  • Smooth Scroll: Set scroll-behavior: smooth on html element
  • Vertical: Default for sidebars and content navigation
  • Horizontal: Use for narrow content or when space is limited
  • Icons: Optional; use when they clarify section type

Accessibility Notes

  • Semantic HTML: Use <nav> element with aria-label
  • Links: Use anchor tags with proper href attributes
  • Active State: Use aria-current="location" on active item
  • Keyboard: Standard link keyboard navigation (Tab to focus, Enter to activate)
  • Focus Visible: Purple outline indicates keyboard focus
  • Skip Links: Consider adding skip link for keyboard users
